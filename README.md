# JavaScript-Basics

## Fundamental and basic concepts of JavaScript

### Formas de ejecutar JavaScript:

- Consola del navegador

- Etiqueta <script> de HTML

```html
<script>
    console.log('Hello World');
</script>
```
- Archivos JS externos
<script src="hello-world.js"></script>

- Node.js

```
node hello-world.js
```
